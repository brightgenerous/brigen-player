brigen-player
=================

### uses

* Scala 2.11.7
* scalafx 8.0.60-R9

### predefined sbt plugins (require sbt 0.13.x):

* sbt-assembly 0.14.1
* sbt-idea 1.6.0
* sbt-eclipse 4.0.0

### usage:

Go to your favourite shell and enter

    sbt compile

generate project files for your favourite IDE

    sbt gen-idea
    sbt eclipse

build jar

    sbt assembly
